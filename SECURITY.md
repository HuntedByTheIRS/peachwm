# Security Policy

PeachWM is a Wayland compositor — a window manager. A major security
vulnerability should never really happen, as it's literally just a
window manager. That said, if you find a flaw, please handle it
responsibly.

## Reporting a Vulnerability

Do **not** open a public issue disclosing the vulnerability.

Instead:

1. Send an email to **huntedbyth3irs@gmail.com** describing the issue.
2. Open a public issue noting that you have sent an email about a security
   concern, but **do not** include any technical details about the
   vulnerability in the issue body.
